Welcome! to this weather search website

This webpage is used for looking up current weather. In the top of this page there are links to different pages, We are currently in the home page. This page will be instructional on how to use the rest of this website

If you go to the city searcher page you will be able to look up the weather by cities. Once you input a city name and hit "submit" you will receive the city name, country name, current temperature in Fahrenheit, Feels like temperature for the day in Fahrenheit, minimum temperature for the day in Fahrenheit, maximum temperature for the day in Fahrenheit, the current humidity perentage, windspeed in miles per hour, an image of the current weather, and the current weather description

If you go to the zip code searcher page you will be able to look up the weather by zipcode. Once you input a zip code and hit "submit" you will receive the city name, country name, current temperature in Fahrenheit, Feels like temperature in Fahrenheit, minimum temperature for the day in Fahrenheit, maximum temperature for the day in Fahrenheit, the current humidity perentage, windspeed in miles per hour, an image of the current weather, and the current weather description

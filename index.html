Welcome to this weather search website!

This webpage is used for looking up current weather. In the top of this page there are links to different pages, We are currently in the home page. This page will be instructional on how to use the rest of this website.

If you go to the city searcher page you will be able to look up the weather by cities. You can input a city name, Country name and hit "submit" you will receive the city name, country name, current temperature, Feels like temperature for the day minimum temperature for the day, maximum temperature for the day in Fahrenheit and Celsius, the current humidity percentage, windspeed in miles per hour, an image of the current weather, and the current weather description.

If you go to the zipcode searcher page you will be able to look up the weather by zipcode. Once you input a zip code and hit "submit" you will receive the city name, country name, current temperature, Feels like temperature for the day, the minimum temperature for the day, maximum temperature for the day in Fahnrenheit Celsius and , the current humidity percentage, wind speed in miles per hour, an image of the current weather, and the current weather description.
